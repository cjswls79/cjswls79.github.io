Research

Grounded in a small set of repeatable rules.

The visual language stays restrained: one dark canvas, one light canvas, a single soft hairline, and no extra accent colors.

Surface

Dark bands create the page rhythm.

Hero and research sections use near-black navy so content changes feel structural.

Type

Display text stays sentence-case.

Labels use uppercase mono; paragraphs and headlines stay readable and direct.

Chrome

Hairlines replace heavy shadows.

Cards sit flat on the canvas with crisp borders and compact internal spacing.

Signal

The ribbon carries the decoration.

The gradient appears once at hero scale instead of becoming scattered UI color.

Work

Current page modules.

Each module is plain HTML and CSS, designed to be easy to edit without adding a framework or package manager to the repository.

Hero

Dark entry with split layout.

The first viewport pairs a direct identity statement with the brand ribbon graphic.

System

Token-driven sections.

Colors, spacing, cards, buttons, and grid behavior are centralized in CSS variables.

Responsive

Stacked mobile flow.

Navigation collapses into a drawer and all grids become one-column on small screens.
